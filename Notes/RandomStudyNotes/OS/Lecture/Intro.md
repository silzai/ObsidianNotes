# Content:
- Every chapter is a function of when the operating system executes something
- process is any program (when loaded into memory) in execution
- main memory management divided into 2:
	- main memory
	- virtual memory
- CPU scheduling, which order to use to run processes
- Inter-process communication 
	- one process' output is another process' input
	- It uses system calls, OS uses a library of functions
- Synchronization/Concurrency
- Multi-threading, 
	- breaking down a big program to make the divisions run simultaneously to speed up execution
- Socket programming over network
